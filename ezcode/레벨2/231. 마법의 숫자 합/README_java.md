# 231. 마법의 숫자 합
- 제출 언어: Java (17)
- 제출 일자: 2026-01-16 07:52:56

## 문제 설명
마법사 리나는 N개의 숫자를 모아 하나의 마법 숫자를 만들려고 합니다.
N개의 숫자가 주어질 때, 이 숫자들의 합을 출력하세요.

### 입력
첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다.

둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다. (각 정수는 -1,000 이상 1,000 이하)

### 출력
숫자들의 합을 출력한다.
